# 36Kr
#Features 
*   解决视频播放器的网速慢卡顿
*   视频播放器调用简单
*   进度条滑动，支持cell全屏和小屏播放
*   代码简洁

#Known problems
*   代码简洁，所有功能都封装在库中你只需要简单调用，支持cell中播放
*   播放器使用的是View + AVPlayer，支持播放mp4、mov等格式，网络和本地视频同时支持,支持Cell中全屏和小屏自动感应旋转屏幕
